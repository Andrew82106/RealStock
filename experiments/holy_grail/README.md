# 圣杯实验（holy_grail）

用韭菜模拟器现场制造并戳破一个"交易圣杯"：

1. 设计一个超级指标：**韭菜共振指数 LRI (Leek Resonance Index)**（`grail_indicator.py`）——
   0-100 振荡器，动量共振核 × 量能激励项 × 玄学修正因子（八日玄学周期，纯编的），
   **≥88 买入，≤44 清仓**；另有裸规则版"追涨杀跌"（`chase_strategy.py`）作对照；
2. 从 2019-12 ~ 2026-07 的历史行情里，搜索让它表现最风光的 22 个交易日；
3. 锁定参数，继续向后运行，看它现出原形。

前端 K 线图的 LRI 副图（`frontend/src/components/StockChart.tsx`）与本目录的
`grail_indicator.py` 使用完全相同的公式和参数。

一段精心挑选的历史高收益，不能证明指标以后还能赚钱。本实验仅供学习与娱乐，不构成任何投资建议。

## 复现步骤

```bash
# 在仓库根目录
pip install -e .
pip install matplotlib

# 1. 搜索高光区间（候选池 × 参数网格 × 全部起始日，结果存 golden_windows.csv）
python experiments/holy_grail/find_golden_window.py

# 2. 对某个高光区间做真实回测复核 + 锁参续跑（结果存 results/）
python experiments/holy_grail/run_golden_and_forward.py 002583 3 5 2024-09-25 2024-10-31
```

真实回测包含：佣金（0.025%，最低5元）、印花税（卖出0.05%）、T+1、
100股整手、一字板当天不成交。

## 2026-07-14 实测结果（10万本金）

| 标的 | 高光区间 | 高光倍数 | 同一账户跑到2026-07 | 利润回吐 | 锁参后才入场 |
|---|---|---|---|---|---|
| 海能达 002583 | 2024-09-25~10-31 | **4.34x** | 43.4万 → 13.4万 | 90% | 10万 → 3.5万 (−65%) |
| 正丹股份 300641 | 2024-04-02~05-15 | 5.11x | 60.6万(峰) → 23.9万 | 73% | 10万 → 4.9万 (−51%) |
| 中通客车 000957 | 2022-05-13~06-21 | 4.04x | 40.7万(峰) → 14.8万 | 84% | 10万 → 4.1万 (−59%) |
| 光智科技 300489 | 2024-09-09~10-24 | 5.76x | **继续涨到 111 万** | 8% | 10万 → 20.7万 (+107%) |

注意最后一行：同一个指标锁参后在光智科技上继续封神。
所以连"拍脑袋指标必死"这个结论，也无法从单个例子得出——这正是回测最危险的地方。

## 数据质量警告

- **腾讯接口的"前复权"数据是减法复权**，低价区间日涨跌幅严重失真
  （九安医疗 2021-11 出现 +39.8% 的"日涨幅"，主板涨停上限只有 10%），
  **绝对不能用于百分比策略回测**。
- 腾讯"后复权"数据为比例复权，日收益率与真实一致。
  本实验通过 `data_loader.RatioAdjustedDataEngine` 使用
  "后复权取数 + 缩放回区间首日真实价位"的方式，保证收益率正确。
- 可用 `check_tx_adjust.py` 和 `check_cache_sanity.py` 自行验证数据质量。

## 已知简化

- 以当日收盘价成交（真实盘中成交价会有差异）；
- 一字板简化为"当天完全无法成交"，非一字的涨跌停板仍按可成交处理；
- 未模拟排队、滑点和流动性限制；
- 股票池为人工挑选的 23 只历史妖股与对照股，本身就是幸存者偏差的一部分——这正是视频要说的事。

## 文件说明

| 文件 | 用途 |
|---|---|
| `chase_strategy.py` | 拍脑袋追涨杀跌策略 |
| `data_loader.py` | 保证收益率正确的数据加载器（后复权+缩放） |
| `find_golden_window.py` | 高光区间搜索 |
| `run_golden_and_forward.py` | 高光复核 + 完整剧情弧 + 锁参后入场 三轮真实回测 |
| `check_tx_adjust.py` | 复权方式数据质量对比 |
| `check_cache_sanity.py` | 缓存数据涨跌幅越限检查 |
| `golden_windows.csv` | 搜索结果 |
| `results/` | 真实回测净值、成交记录、图表 |
